# FundThePitch 系統使用說明（教師版）

> 本文件僅說明系統功能與操作流程，適合教師快速了解與現場使用。

## 1. 系統用途

FundThePitch 是一套專題評分投資系統。
每位評審有固定預算 10,000 元，需分配到各專題，系統會即時彙整結果並產生排名。

## 2. 角色與權限

### 管理員（Admin）
- 啟動與關閉本次活動
- 管理評審帳號
- 管理評分組別與專題清單
- 查閱歷史封存紀錄
- 協助評審解鎖重填

### 評審（Judge）
- 以本名作為登入帳號登入系統
- 進入指定評分組別
- 對專題分配投資金額
- 可先暫存草稿，再鎖定提交

### 投影觀戰帳號（僅查看）
- 此帳號僅用於投影畫面，即時查看戰況。
- 不需要加入評分組別。
- 不可提交投資、不可修改資料。
- 建議固定停留在 Dashboard 畫面供現場觀看。

## 3. 主要畫面功能

### A. 大廳（Lobby）
- 使用者登入入口
- 評審可選擇要進入的評分組別
- 可切換至儀表板與其他功能頁

### B. 評審投資頁（Judge UI）
- 顯示本輪可投資的專題清單
- 以數字輸入或滑桿調整各專題金額
- 顯示「已分配 / 剩餘」預算
- 兩種提交方式：
  - 暫存：先存草稿，之後可再修改
  - 鎖定上傳：正式送出後不可自行修改

### C. 即時儀表板（Dashboard）
- 即時顯示各專題累積投資金額
- 顯示各評審的個別投資明細
- 提供簡報模式：逐步揭曉排名，適合現場展示
- 支援投影觀戰帳號長時間顯示，不需進入評分流程

### D. 管理面板（Admin）
- 活動流程控制：啟動、關閉、封存
- 評審管理：新增、調整、解除鎖定
- 組別與專題設定：建立、修改、刪除
- 歷史資料查詢：回看過往場次結果

## 4. 標準操作流程（建議）

### 活動前
1. 管理員先建立本次活動。
2. 設定評分組別與每組專題。
3. 建立評審名單並確認可登入。

### 活動中
1. 評審以本名登入後進入自己的評分組別。
2. 先使用「暫存」逐步填寫。
3. 確認總額正確後，按「鎖定上傳」。
4. 投影觀戰帳號登入後，直接開啟 Dashboard 即時展示戰況。
5. 現場可同步用儀表板做結果播報與排名揭曉。

### 登入說明（教師口頭提醒建議）
- 評審登入帳號為本人姓名（本名）。
- 若姓名輸入錯誤，可能無法進入正確評分權限。

### 活動後
1. 管理員關閉本次活動。
2. 系統自動封存本次結果。
3. 可於管理面板查閱歷史紀錄。

## 5. 投資規則重點

- 每位評審預算固定為 10,000 元。
- 鎖定上傳時，總額必須符合系統規則。
- 一旦鎖定，評審不可自行改動；需由管理員解鎖。

## 6. 教師現場建議

- 先安排 3 至 5 分鐘讓評審熟悉操作。
- 宣布「先暫存、後鎖定」流程，降低誤送風險。
- 由管理員統一宣布截止後再進入簡報模式。
- 若有評審操作失誤，請管理員協助解鎖後重提。

## 7. 常見問題（FAQ）

### Q1：評審看不到提交按鈕或無法送出？
- 請先確認是否已登入正確角色。
- 檢查預算是否符合提交規則。
- 若已鎖定，需由管理員解鎖。

### Q2：儀表板數據沒有更新？
- 確認評審是否已完成提交。
- 重新整理儀表板頁面。
- 確認後端服務正常運行。

### Q3：評審要修改已送出的內容怎麼辦？
- 由管理員在管理面板執行解除鎖定。
- 評審重新進入投資頁調整後再鎖定上傳。
